- главное окно теперь можно сжимать до 640x460;
- sidebar получил компактный режим:
  - на узком окне остаются иконки;
  - текст скрывается;
  - ширина уменьшается до 74 px;
- topbar получил компактный режим:
  - меньше высота;
  - скрывается статус READY;
  - кнопка Играть становится компактной;
  - поиск сжимается;
- MainWindow автоматически включает компактный режим при ширине < 900 px;
- вкладка Аккаунты полностью переведена в ScrollArea;
- Аккаунты адаптируются:
  - stats-карточки: 4/2/1 колонка;
  - провайдеры: 3/1 колонка;
  - профили и скины: 2 колонки на широком окне, 1 колонка на узком;
- добавлен локальный OAuth callback server:
  - `auth/oauth_callback_server.py`;
  - больше не нужно вручную вставлять redirect URL;
  - Nexus открывает браузер, ловит localhost callback и завершает вход сам;
- Microsoft OAuth:
  - автозахват callback;
  - сохранение access_token в аккаунте;
  - сохранение refresh_token в tokens;
  - аккаунт автоматически становится активным;
- Ely.by OAuth:
  - автозахват callback;
  - сохранение access_token в аккаунте;
  - сохранение refresh_token в tokens;
  - аккаунт автоматически становится активным;
- провайдеры нормализованы:
  - `microsoft`
  - `ely`
  - `offline`
- сохранены все предыдущие улучшения:
  - моды/модпаки/шейдеры/ресурспаки;
  - .mrpack importer;
  - Update Center;
  - Проверка сборки;
  - Статус “Установлено”;
  - фикс Подробнее;
  - адаптивность остальных страниц.

Важно:
- Microsoft вход требует настоящий Azure App Client ID и redirect URI.
- Ely.by вход требует OAuth-приложение Ely.by и client secret.
- Без этих данных приложение не может легально и безопасно логинить пользователя за разработчика.
