# Nexus Content Split + Shader Install Fix

Что исправлено:

1. Шейдеры:
- shader projects теперь устанавливаются именно в `.minecraft/shaderpacks`;
- resource packs — в `.minecraft/resourcepacks`;
- mods — в `.minecraft/mods`;
- если Modrinth отдаёт `project_type`, Nexus использует его при установке, а не только выбранный фильтр;
- зависимости shader/resourcepack больше не ставятся как shader/resourcepack, чтобы Iris `.jar` случайно не попадал в `shaderpacks`;
- после установки шейдера сообщение объясняет, что Iris Shaders — это загрузчик, а shader pack — отдельный `.zip`.

2. Разделение контента:
- сверху на странице «Моды» добавлены постоянные вкладки:
  - Моды
  - Модпаки
  - Шейдеры
  - Ресурспаки
- установка теперь автоматически идёт в правильную папку по типу проекта;
- на карточках показывается тип проекта.

3. Фильтры:
- выбор типа/категории/стороны/сортировки запускает поиск;
- для shader/resourcepack фильтр стороны автоматически отключается;
- shortcut-карточки больше не сворачивают фильтры после клика.

4. Светлая тема:
- временно отключена;
- кнопка темы теперь переключает только dark ↔ AMOLED;
- если в настройках была сохранена `light`, Nexus автоматически вернёт `dark`.

Проверка:
1. Открыть «Моды».
2. Нажать вкладку «Шейдеры».
3. Найти `Complementary Reimagined` или `MakeUp Ultra Fast`.
4. Нажать «Установить шейдер».
5. Проверить папку `.minecraft/shaderpacks` — там должен быть `.zip`.
