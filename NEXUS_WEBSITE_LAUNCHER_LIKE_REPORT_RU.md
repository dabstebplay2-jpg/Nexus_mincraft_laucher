# Nexus Website Launcher-Like Redesign

Сайт приближен к визуальному стилю лаунчера.

Что изменено:
- убрана гигантская лендинговая типографика;
- добавлена app-like структура: header-shell, preview с левым sidebar, Minecraft hero, карточки;
- цвета приведены к лаунчеру: тёмный камень, мох, трава, приглушённый зелёный;
- download-блок сделан как панель приложения;
- preview теперь похож на окно Nexus Launcher;
- сохранены прямые ссылки на GitHub Release;
- JS автоматически подхватывает latest release и обновляет номера версии/ссылки.

- website/styles.css
- website/style.css
- website/script.js
- website/app.js
