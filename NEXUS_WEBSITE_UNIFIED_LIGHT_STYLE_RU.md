# Nexus Website Unified Light Style

Исправлена проблема смешения двух стилей.

Проблема:
- общий сайт был светлый и мягкий;
- правый Download Center был тёмный и выглядел как отдельный чужой блок;
- секция comparison тоже имела тёмные карточки и выбивалась из общей стилистики.

Что изменено:
- Download Center переведён в тот же светлый app-store стиль;
- release row, asset cards, meta rows и copy button приведены к общей палитре;
- comparison cards теперь светлые;
- убраны резкие тёмные контрасты;
- сохранена структура сайта и GitHub latest release integration.

Изменены файлы:
- website/index.html
- website/styles.css
- website/style.css
