# Nexus Launcher — Mods & Downloads Final Upgrade

Фокус обновления:
- моды больше не скачиваются бесконечно повторно;
- кнопка в каталоге показывает состояние установки;
- установка блокируется, если мод уже есть в выбранной сборке;
- если запись о моде есть, но файл потерян, Nexus предлагает переустановить;
- installer проверяет совместимую версию по Minecraft + loader;
- после установки сохраняются версия, loader, Minecraft version, source, files, file_names, icon_url и installed_at;
- при обновлении/переустановке старый jar этого же проекта удаляется после успешной загрузки;
- страница загрузок стала подробной: размер, скорость, источник, длительность, даты, прогресс;
- DownloadManager теперь хранит speed_bps;
- страницы модов/загрузок/сборок/библиотеки отключают горизонтальный scrollbar;
- добавлен общий модуль `mods/mod_status.py` для определения состояния мода.

- проект проверен через py_compile;
