Кратко

Nexus Launcher — локальное desktop-приложение. Сайт nexus-launcher.github.io не использует cookies, не собирает персональные данные и не содержит сторонней аналитики.

Что делает лаунчер

  • Хранит сборки, настройки и логи локально на вашем компьютере (%APPDATA%\NexusLauncher).
  • Обращается к API Mojang, Modrinth и другим сервисам только для скачивания игры и модов.
  • Не отправляет ваши файлы на серверы Nexus — у проекта нет собственного backend.

Что делает сайт

  • Отдаёт статические файлы (HTML, CSS, JS) и архив для скачивания.
  • Не регистрирует аккаунты и не запрашивает email.
  • Хостинг-провайдер (GitHub Pages, Netlify и т.д.) может вести стандартные server logs — см. их политику.

Контакты

По вопросам проекта используйте репозиторий GitHub проекта Nexus Launcher.
