Скачать

Одна кнопка — прямой download установщика.

Кнопка скачивает Setup EXE из последнего GitHub Release без перехода на страницу GitHub. После установки Nexus запускается как обычное Windows-приложение.

FAQ

Частые вопросы

Это официальный лаунчер Mojang/Microsoft?

Нет. Nexus Launcher — отдельный неофициальный проект.

Скачивание идёт без перехода на GitHub?

Да. Кнопка ведёт прямо на asset из последнего GitHub Release.

Куда ставится программа?

В папку пользователя Windows, с ярлыком на рабочем столе после установки.
