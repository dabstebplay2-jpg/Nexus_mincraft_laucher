# Nexus Launcher — Calm Compact UX Update

Что изменено:

## 1. Общий стиль лаунчера
- Убрана слишком агрессивная неоновость.
- Палитра стала спокойнее: тёмный графит + мягкий зелёный акцент.
- Все карточки и панели приведены к более ровному и аккуратному виду.
- Кнопки, поля ввода и элементы состояния стали компактнее.

## 2. Sidebar / левая навигация
- Сайдбар стал уже и аккуратнее.
- Названия разделов упрощены.
- Компактный режим теперь включается на более широком окне, чтобы интерфейс не распирало.
- Логотип и профиль визуально спокойнее и чище.

## 3. Topbar / верхняя панель
- Верхняя панель стала ниже и легче.
- Убрана лишняя шумная кнопка статуса.
- Оставлены только реально полезные элементы:
  - название страницы,
  - поиск,
  - переключение темы,
  - кнопка "Играть".

## 4. Главная страница полностью переработана
Старый экран был перегружен. Теперь главная разбита на 4 понятных блока:

- **Hero-блок** — короткое объяснение, что делает Nexus.
- **Краткая сводка** — сборки, моды, загрузки, статус.
- **Последние сборки** — быстрый доступ к актуальным инстансам.
- **Быстрый старт** — пошаговые основные действия.
- **Недавние действия** — последние загрузки.
- **Система и лаунчер** — RAM, свободная память, Java и память Minecraft.

## 5. Цель обновления
Главная цель этого патча — чтобы пользователь:
- не терялся сразу после запуска,
- видел только главное,
- быстрее понимал, куда нажимать,
- не получал визуальную перегрузку.

## Какие файлы изменены
- `app/window.py`
- `ui/styles.py`
- `ui/components/sidebar.py`
- `ui/components/topbar.py`
- `ui/pages/home_page.py`

## Как применить patch ZIP
Открой корень проекта и распакуй patch ZIP **поверх проекта с заменой файлов**.

## После распаковки запусти
```powershell
cd C:\Nexus
.\.venv\Scripts\python.exe main.py
```

Если `.venv` отсутствует — сначала создай окружение и установи зависимости так, как ты уже делал раньше.
