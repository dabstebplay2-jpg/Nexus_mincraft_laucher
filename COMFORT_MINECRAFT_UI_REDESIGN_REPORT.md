# Nexus Launcher — Comfort Minecraft UI Redesign

Сделано по замечаниям:
- интерфейс стал компактнее: меньше крупные заголовки, карточки, отступы и кнопки;
- скрыта отдельная кнопка «Аккаунты» в боковом меню;
- профиль игрока внизу sidebar теперь открывает страницу аккаунтов/скинов;
- профиль подсвечивается, когда открыта страница аккаунтов;
- добавлена кнопка смены темы в верхнюю панель;
- тема сохраняется в `data/launcher_settings.json`;
- настройки темы синхронизируются с combobox в «Настройках»;
- светлая/тёмная/AMOLED тема применяются сразу без перезапуска;
- домашняя страница стала компактнее:
  - hero ниже;
  - «Мои сборки» больше не растягивается огромной пустой коробкой;
  - быстрые действия сделаны в 2 колонки;
  - убран отдельный quick action «Аккаунты»;
- вкладка «Моды» стала интереснее:
  - добавлены Minecraft shortcut-карточки: FPS Boost, Шейдеры, Vanilla+, Ресурспаки;
  - фильтры перенесены в адаптивную сетку, чтобы не появлялся горизонтальный скролл;
  - статистика перенесена в 2 ряда вместо длинной строки;
  - карточки модов стали компактнее;
- общий QSS получил комфортный Minecraft-style patch;
- горизонтальный scrollbar визуально убран;
- сайт стал спокойнее:
  - меньше огромных заголовков;
  - меньше пустых расстояний;
  - компактнее hero/app preview/cards;
  - сохранён Minecraft/glass стиль, но без перегруза;
- версия поднята до `0.7.3`.
