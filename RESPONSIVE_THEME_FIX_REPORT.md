# Nexus Launcher — Responsive + Theme Fix

Исправлено после замечаний:
- главный экран теперь находится внутри вертикального ScrollArea, поэтому блоки не ломают высоту окна;
- sidebar стал уже, чтобы контенту оставалось больше места;
- topbar search стал гибким по ширине;
- главное hero-окно больше не вываливает RAM/Java карточки за границы: статистика перестроена в 2x2;
- быстрые действия на главной больше не режутся: карточки идут одной колонкой;
- сетка модов стала адаптивной: 1 колонка на узком окне, 2 колонки на широком;
- главное окно стало более гибким по минимальному размеру;
- исправлена ошибка отсутствующего `data/launcher_settings.json`;
- тема теперь сохраняется безопасно даже если файла настроек ещё нет;
- тема применяется сразу без обязательного перезапуска через `MainWindow.apply_theme`;
- настройки теперь подтягивают сохранённую тему и язык обратно в combo-box.

Проверка:
- изменённые Python-файлы проходят `py_compile`;
- синтаксических ошибок нет.
