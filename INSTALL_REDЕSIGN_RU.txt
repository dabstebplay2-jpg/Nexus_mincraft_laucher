Как запустить Nexus Launcher:\n\n1. Распакуй архив в удобную папку.\n2. Открой PowerShell в папке проекта.\n3. Если у тебя уже есть .venv в старом проекте — можешь скопировать туда эти файлы с заменой.\n4. Если запускаешь как новый проект:\n   python -m venv .venv\n   .\\.venv\\Scripts\\python.exe -m pip install -r requirements.txt\n   .\\.venv\\Scripts\\python.exe main.py\n\nВажно: для текущего рабочего проекта проще скопировать содержимое архива в C:\\Nexus_minecraft_launcher с заменой файлов.\n
